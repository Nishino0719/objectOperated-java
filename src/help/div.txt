## divコマンドについて

 逆行列との積を利用することで除算の動きをする「コマンド」. 
 AX = B
 においてXの部分を求めるコマンド
 A...入力した行列
 B...現在の結果の行列
 div :
  TAB  a_11 ... a_1m
  TAB  a_21 ... a_2m
    ...
  TAB  a_n1 ... a_nm
 という, 1行目が {@code div} である複数行「ブロック」を受け付け, 入力された行列の逆行列と現在の行列の積の「結果」を返す. 
 行列の入力は, 各行の要素を TAB 始まりの各行に空白区切りで並べて入力する. 
 例えば, 次のような「ブロック」を入力として受け付ける（2行目以降は TAB を先頭に入力すること）. 
 また,現在の行列の列と行、入力した列と行の数が一致している必要がある.
# example
 div :
      1 0 1
      0 1 0
      0 1 1
 もしくは
 div x
 とすることによって変数に保存された行列をかけた結果を返す.
