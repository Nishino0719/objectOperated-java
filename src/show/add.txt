## addコマンドについて

 行列加算の「コマンド」. 
 add :
  TAB  a_11 ... a_1m
  TAB  a_21 ... a_2m
    ...
  TAB  a_n1 ... a_nm
 という, 1行目が {@code add} である複数行「ブロック」を受け付け, 入力された行列を足した「結果」を返す. 
 行列の入力は, 各行の要素を TAB 始まりの各行に空白区切りで並べて入力する. 
 例えば, 次のような「ブロック」を入力として受け付ける（2行目以降は TAB を先頭に入力すること）. 
# example
 add :
      1 0 1
      0 1 0
 もしくは
 add x
 とすることによって変数に保存された行列を足した結果を返す.
